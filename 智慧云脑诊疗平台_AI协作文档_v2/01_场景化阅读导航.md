## 3. 角色快速路径

### 后端 AI

```text
后端任务书
→ 当前后端子任务
→ 相关业务流程
→ 仓库根 contracts/openapi.yaml 对应接口
→ 数据库设计基线
→ 数据库与 Flyway 规范
→ 模块边界
```

接口或字段不满足需求时，不修改 `contracts/`，提交变更申请。

### 前端 AI

```text
前端任务书
→ 当前前端子任务
→ 仓库根 contracts/openapi.yaml 对应接口
→ 相关业务流程和页面验收
→ 前端工程规范
```

接口缺字段时，不自行兼容多个字段名，不修改 OpenAPI。

### AI 能力集成 AI

```text
AI 任务书
→ 当前 AI 子任务
→ AI 能力契约
→ 相关业务流程和确定性规则
→ Prompt、Mock 和降级要求
```

需要新增 Maven 依赖时，将依赖说明交给后端 AI。

### 联调测试与集成 AI

```text
联调任务书
→ 当前父任务和联调子任务
→ 仓库根 contracts/openapi.yaml
→ 相关角色子任务
→ 测试与完成定义
→ PR 与交付规范
```

发现实现问题时分派给责任角色，不在业务目录做临时兼容。

### 整体架构设计 AI

```text
架构任务
→ 整体架构
→ 模块与依赖边界
→ 项目目录结构
→ 目录所有权
→ 受影响角色任务书
```

只给出结构和协作约束；具体实现由对应开发角色完成。
