# Frontend Modules

业务模块只能在对应任务卡进入 `READY` 且契约明确后新增实现。当前目录仅保留角色与业务域的所有权边界。

- `patient/`: 患者端
- `doctor/`: 医生端
- `admin/`: 管理端
- `auth/`: 认证
- `schedule/`: 排班
- `appointment/`: 预约
- `triage/`: 分诊
- `encounter/`: 就诊
- `examination/`: 检查
- `medical-record/`: 病历
- `prescription/`: 处方
- `device/`: 设备
- `statistics/`: 统计
